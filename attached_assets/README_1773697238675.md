# JobSearch Pro v2.0

**Menu-driven · Resume-aware · Multi-profile · Multi-source**

Searches Indeed, LinkedIn, ZipRecruiter, Glassdoor, and Adzuna daily for
remote/hybrid jobs matched to your resume skills. Builds a formatted Excel
report, uploads to your NAS, and emails you the file.

---

## Quick Start — Local / Agent Zero

```bash
# 1. Install everything
chmod +x setup.sh && ./setup.sh

# 2. Launch
./run.sh
```

**First-time flow inside the app:**
1. `Manage Search Profiles` → `Create new profile`
2. `Upload & Parse Resume` → upload your `.pdf` or `.docx`
   - System extracts your titles, skills, and builds search queries
   - Review and edit the suggested queries before saving
3. `Configure Output & NAS` → set your NAS share path and credentials
4. `Configure Email` → add your SMTP / Gmail App Password
5. `Run Job Search` → launches the full search

## Quick Start — Replit

1. Create a new Replit → **Import from GitHub** or paste all files
2. The run button automatically uses `replit_main.py`
3. In the **Shell** tab: `pip install -r requirements.txt`
4. Hit **Run** — the menu launches in the console
5. After a search run, find your `.xlsx` in **Files → output/**

> NAS upload is disabled on Replit (no local network). Everything else
> works identically — profiles, resume parsing, email, spreadsheet.

## Resume Parsing

The parser extracts from your `.pdf` or `.docx`:

| Extracted | Examples |
|---|---|
| Job titles held | Procurement Manager, Subcontracts Administrator |
| Systems / Tools | SAP, Ariba, ERP, P2P, Coupa, MS Office |
| Procurement skills | Strategic sourcing, RFP/RFQ, vendor management |
| Contract skills | FAR, DFARS, subcontracts, government contracting |
| Certifications | Lean Six Sigma, Green Belt, PMP, P&C License |
| Industries | Defense, Aerospace, Healthcare, SaaS, Government |

From those signals it generates tailored search queries like:
- `subcontracts manager defense remote`
- `procurement manager SAP remote`
- `contract administrator FAR DFARS remote`
- `operations manager lean six sigma remote`

The **approve/edit screen** lets you keep, modify, or remove each suggested
query before it's added to your profile.

## Profiles

Each named profile stores independently:
- Salary minimum
- Work type filter (Remote / Hybrid / Both)
- Which sources to search
- Its own list of search queries
- Resume source reference

You can have multiple profiles — e.g.:
- `"Defense Contracts"` — FAR/DFARS queries, defense sources, $95K min
- `"Healthcare Remote"` — healthcare vendor/ops queries, $80K min
- `"Tech SaaS"` — tech operations/procurement, all sources

Switch the active profile from **Manage Profiles → Select active profile**.

## NAS Setup

### Option A — Script mounts on each run (default)

Add passwordless sudo for mount in `/etc/sudoers`:
```
yourname ALL=(ALL) NOPASSWD: /bin/mount, /bin/umount
```

### Option B — Permanent fstab mount (recommended)

Add to `/etc/fstab`:
```
//192.168.1.XXX/SHARE  /mnt/nas_jobs  cifs  credentials=/home/you/.nas_creds,vers=3.0,uid=you,gid=you,file_mode=0664,dir_mode=0775,_netdev,nofail  0  0
```

Create `/home/you/.nas_creds`:
```
username=NAS_USER
password=NAS_PASSWORD
domain=WORKGROUP
```
```bash
chmod 600 ~/.nas_creds
sudo mount -a
```

In the app: **Output & NAS → NAS enabled → Yes**, then set
`NAS_USE_MOUNT = False` and point to `/mnt/nas_jobs`.

## Gmail App Password

Gmail requires an App Password — not your regular password.
1. Go to https://myaccount.google.com/security
2. Enable 2-Step Verification
3. Search "App Passwords" → Create → name it "JobSearch Pro"
4. Copy the 16-character code into **Configure Email → SMTP password**

## Troubleshooting

**Few or no results returned**
Job boards change their HTML structure periodically. The scrapers use
multiple CSS selector fallbacks, but if a site redesigns significantly,
selectors may need updating. Check `logs/job_scraper.log` — lines marked
`[source] 'query' → 0 jobs` indicate which source/query to investigate.

**NAS mount fails**
```bash
# Test manually
sudo mount -t cifs //NAS_IP/SHARE /mnt/test -o username=USER,password=PASS
# Check cifs-utils is installed
sudo apt install cifs-utils
```

**Email not sending (Gmail)**
Confirm you are using an **App Password** (16 chars, no spaces), not your
Gmail account password. Test: **Configure Email → Test email**.

# REPLIT PROMPT
## (paste this into Replit AI / Ghostwriter or use as a project description)

```
Build and run JobSearch Pro — a Python terminal application that searches
multiple job boards for remote and hybrid jobs, parses the user's resume to
build targeted search queries, and exports results to a formatted Excel file.

PROJECT STRUCTURE ALREADY PROVIDED:
  main.py, replit_main.py, requirements.txt, .replit
  core/: config_manager.py, profile_manager.py, resume_parser.py,
         scraper.py, spreadsheet.py, nas_uploader.py, emailer.py
  ui/menu.py

ENTRY POINT FOR REPLIT:
  python replit_main.py

INSTALL DEPENDENCIES:
  pip install -r requirements.txt

WHAT THE APP DOES:
  1. Presents a numbered terminal menu
  2. Lets the user create named "search profiles" with custom job queries,
     salary minimums, work type filters (remote/hybrid), and sources
  3. Parses uploaded PDF or DOCX resumes to extract job titles, skills,
     tools, certifications, and industries — then generates targeted search
     queries from those signals
  4. Shows the user the extracted skills and suggested queries for approval
     and editing before saving them to the profile
  5. Searches Indeed, LinkedIn, ZipRecruiter, Glassdoor, and Adzuna using
     BeautifulSoup scraping
  6. Deduplicates results and builds a color-coded .xlsx spreadsheet with
     three tabs: Job Listings, Summary, and Search Config
  7. Optionally emails the spreadsheet (configure SMTP in the menu)
  8. Saves all output to the output/ directory; user downloads from
     Replit's Files panel

REPLIT-SPECIFIC NOTES:
  - NAS upload is disabled (no local network in Replit cloud)
  - All file output goes to output/ in the Replit filesystem
  - The .replit config sets replit_main.py as the run target
  - No web server or Flask needed — this is a pure terminal/console app
  - The menu uses ANSI color codes; these render correctly in Replit's console

IF THE USER ASKS TO MODIFY THE APP:
  - To add a new job source: add a method _sourcename() to core/scraper.py
    following the same pattern as _indeed(), then add it to the dispatch dict
  - To add preset queries: edit the PRESETS dict in ui/menu.py _add_presets()
  - To add new resume skill detection: add keywords to SKILL_KEYWORDS in
    core/resume_parser.py and add templates to QUERY_TEMPLATES
  - To change spreadsheet formatting: edit core/spreadsheet.py

TESTING WITHOUT REAL SCRAPING:
  To test the menu and spreadsheet without hitting live job boards, add a
  --mock flag: in core/scraper.py search() method, if os.environ.get('MOCK'):
  return a list of 5 fake job dicts instead of calling the real scraper.
```
